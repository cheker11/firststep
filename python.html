Python як мова програмування

Python — високорівнева мова програмування, яку називають другою за популярністю в світі. Її використовують для розробки вебзастосунків, програмного забезпечення, машинного навчання. Інтерпретованість дозволяє знаходити помилки в коді ще до повної збірки в робочий застосунок. При цьому Python дуже чітко дає зрозуміти, де та через що виникла помилка. У даної мови логічний синтаксис, завдяки чому вихідний код легко читати та розуміти. Програми, написані на мові програмування Python, можуть бути як невеликими скриптами, так і складними системами.

Чотири галузі, де можуть працювати Python-розробники

1. Web-розробник

Python-розробник займається створенням внутрішньої логіки та архітектури вебдодатку, сторінки, допомагає їй взаємодіяти з базами даних та сервером.

2. Штучний інтелект та машиннє навчання

Python-розробники беруть активну участь в програмуванні систем штучного інтелекту. Одним із напрямів їхньої роботи є машинне навчання - процес, під час якого комп’ютер обробляє величезну кількість даних та на їх основі навчається «приймати» рішення.

3. Data Science

Python-девелопер, який розуміється на Data Science, може створювати потоки даних для навчання машин або працювати аналітиком.

4. GameDev

Python розробник може створювати ігри. Мова програмування Python необхідна для прописування ігрової логіки, програмування серверних елементів, написання скриптів.

Базові команди в Python

input

Використовується для обробки введення користувача в python. Обертаючи input() функцією int(), ми перетворимо введений рядок у ціле число.Якщо вам необхідно ввести дійсне число і зберегти його в змінну, необхідно обернути функцію input() на float()

print

Функція дозволяє вивести текст, значення змінних та інші дані в консоль для аналізу чи візуалізації результатів програми. Для того щоб це зробити, після print ставляться дужки і лапки. Виглядає це приблизно так: print("текст")

Вибір Програми (IDE)

Перед написанням коду треба вибрати програму. Кожен підбирає її індивідуально під себе. В нашому курсі буде використана програма PyCharm, але розглянемо плюси й інших застосунків.

PyCharm

Одне з найпопулярніших безкоштовних інтегрованих середовищ розробки (IDE).

  • ☑ Вбудоване налагодження та тестування (полегшує відстеження та виправлення помилок)
  • ☑ Вбудована підтримка PEP8 (автоматична перевірка коду на відповідність стандартам)
  • ☑ Інтелектуальне автозаповнення (включає розуміння контексту коду, автоматичне виявлення помилок)

Visual Studio Code

Безкоштовне відкрите середовище розробки від Microsoft.

  • ☑ Підтримка розширень (дозволяє налаштовувати середовище розробки під свої потреби)
  • ☑ Вбудована підтримка Git (GIT – це система, що спрощує роботу програмістів над різними проектами)
  • ☑ Розширення Python (надає безліч функцій для розробки на Python, включаючи форматування коду)

Spyder

Відкрите IDE спеціально розроблене для мови програмування Python.

  • ☑ Багатофункціональний редактор з підсвічуванням синтаксису, автозаповненням коду
  • ☑ Повнофункціональний відладчик для пошуку та виправлення помилок
  • ☑ Переглядач змінних (функція дозволяє переглядати та редагувати значення змінних, що використовуються в коді)

Atom

Відкрите IDE створене GitHub.

  • ☑ Інтегрована файлова система (дозволяє легко переглядати, відкривати та редагувати файли)
  • ☑ Підтримка пакетів (Atom має багато пакетів, створених для розробки на Python)
  • ☑ Вбудований Git та GitHub (Ви можете створювати коміти, перевіряти статус свого репозиторія, клонувати репозиторії)

Початок роботи. Калькулятор на Python

Відкриваємо програму та створюємо файл. (File -> New, за потреби створіть папку на робочому столі для файлів, де вони будуть зберігатись). Вибираємо Python File та даємо йому назву. Після цього він буде збережений або в папці, яку ви створили, або в іншій папці вашого IDE.

У першому рядку маємо запитати в користувача про дію, яку буде виконувати програма: додавання, віднімання, множення тощо. Для цього використовуємо функцію input(), а всередину дужок пишемо питання, яке буде виводитись користувачеві. Введення користувача зберігається у змінній what, в неї може бути довільна назва. Далі треба дізнатись числа, з якими будуть відбуватись дії, і для цього використовуємо функцію int(input()) для чисел і змінні a,b.

Починаємо з умовного оператора. Перша дія перевіряє, чи будемо ми робити дію додавання, і якщо так, додаємо перше і друге значення, після чого виводимо 'c' (a+b) рядком (str). Інша дія у нас ділення. Починаємо вже з elif, адже if може бути тільки один раз, як і else, яким ми закінчимо, усі інші - elif. Копіюємо і робимо такі самі дії, як і з додаванням, але змінюємо арифметичні знаки. Коли всі дії прописані, закінчуємо код через else. Він виведе помилку, якщо користувач ввів дію неправильно.

Запускаємо програму (Права клавіша миші -> Run або Ctrl+Shift+F10). Як бачимо, програма рахує наші прикладі і видає результат. Вітаю з вашим першим калькулятором на Python!
